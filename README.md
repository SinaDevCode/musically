# Musically

A music page built with HTML, SASS, and JavaScript

### Design Preview

![Design preview for the Musically](./design/desktop-perview.png)

### Features

-   Listen to music
-   Download music
-   Like and interact with music

### Technologies Used

-   HTML5
-   Sass
-   Javascript
-   Swiper.js
